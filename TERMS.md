## 2. Description of Service

The Service provides an MCP (Model Context Protocol) connector that
answers questions about publicly available SAP Security Note metadata
(Patch Day releases, CVSS scores, CVE IDs, components, release dates,
and CISA KEV exploitation status). The Service serves a static public
catalog; it does not connect to any SAP system.

version applicability before acting.
